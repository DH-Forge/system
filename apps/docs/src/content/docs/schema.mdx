This API provides versioned JSON Schemas for the DH Forge System (DHFS). DHFS is an open standard for Daggerheart player character data, designed to promote interoperability between different tools and applications.

You can learn more about the DH Forge System on [GitHub](https://github.com/dh-forge-system) or by reading the [documentation](https://github.com/dh-forge-system/dh-forge-system/blob/main/README.PRD.md).

## Example Usage

With hand-written JSON, or via your favorite JSON Schema tools.

<Code
	code={JSON.stringify({$schema: "https://dh-forge.com/schema/1.0.x/character.json",name: "John Doe",level: 1,class: "Sorcerer",
	},null,2,)}
	lang="json"
	title="minimal-example.json"
	mark={["$schema"]}
/>
